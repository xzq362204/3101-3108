i2chid_read_fwid v0.0.
--------------------------------------
SYNOPSIS:

[PID]
-p <pid in decimal>.
Ex: i2chid_read_fwid -p 1842
-P <PID in hex>.
Ex: i2chid_read_fwid -P 732 (0x732)

[FWID Mapping Table]
-f <fwid_mapping_table_file_path>.
Ex: i2chid_read_fwid -f fwid_mapping_table.txt
Ex: i2chid_read_fwid -f /tmp/fwid_mapping_table.txt

[System Platform]
-s <system>.
Ex: i2chid_read_fwid -s chrome
Ex: i2chid_read_fwid -s windows

[Device Information]
-i.
Ex: i2chid_read_fwid -i

[Debug]
-d.
Ex: i2chid_read_fwidp -d

[Help]
-h.
Ex: i2chid_read_fwid -h
